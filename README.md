# Portfolio

Portfólio pessoal desenvolvido com Django para apresentar projetos, tecnologias, formação acadêmica, cursos e informações de contato.

Este projeto foi criado com o objetivo de praticar:

* Python
* Django
* Estrutura MVT
* CRUD com Painel Administrativo
* HTML, CSS e JavaScript
* Deploy com Render

## Funcionalidades

O sistema oferece as seguintes funcionalidades:

### Gerenciamento de Projetos

Permite cadastrar, editar e remover projetos através do painel administrativo.

### Gerenciamento de Tecnologias

Exibe as tecnologias utilizadas e permite atualização dinâmica pelo Admin.

### Gerenciamento de Cursos

Permite adicionar e organizar cursos e certificações.

### Área Sobre Mim

Apresenta informações profissionais e objetivos de carreira.

### Contato

Disponibiliza formas de contato e redes profissionais.

### Painel Administrativo

Gerenciamento completo do conteúdo através do Django Admin.
